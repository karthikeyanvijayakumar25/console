SAM V71 Xplained Ultra - Board

Description

Purpose

This file is dedicated to describe the SAM V71 Xplained Ultra board.

Contents

  • For SAM V71 Xplained Ultra board information, see "SAM V71 Xplained Ultra - Board informations".
  • For operating frequency information, see "SAM V71 Xplained Ultra - Operating frequencies".
  • For using portable PIO definitions, see "SAM V71 Xplained Ultra - PIO definitions".
  • For using GMAC PIO definitions, see "SAM V71 Xplained Ultra - GMAC".
  • For using ISI definitions, see "SAM V71 Xplained Ultra - ISI".
  • For on-board memories, see "SAM V71 Xplained Ultra - Memories".
  • Several USB definitions are included here, see "SAM V71 Xplained Ultra - USB device".
  • For External components, see "SAM V71 Xplained Ultra - External components".
  • For Individual chip definition, see "SAM V71 Xplained Ultra - Individual chip definition".

To get more software details and the full list of parameters related to the SAM V71 Xplained Ultra board configuration, please have a look at the source file: board.h, board_v71_xult.h

Usage

  • The code for booting the board is provided by startup_samv71.c and board_lowlevel.c.
  • For using board PIOs, board characteristics (clock, etc.) and external components, see board.h, board_v71_xult.h.
  • For manipulating memories, see board_memories.h.

This file can be used as a template and modified to fit a custom board, with specific PIOs usage or memory connections.
