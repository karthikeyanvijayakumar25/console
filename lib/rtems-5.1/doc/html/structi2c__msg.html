Field Documentation

◆ addr

uint16_t i2c_msg::addr

The slave address.

In case the I2C_M_TEN flag is set, then this is a 10-bit address, otherwise it is a 7-bit address.

◆ flags

uint16_t i2c_msg::flags

The message flags.

Valid flags are

  • I2C_M_TEN,
  • I2C_M_RD,
  • I2C_M_STOP,
  • I2C_M_NOSTART,
  • I2C_M_REV_DIR_ADDR,
  • I2C_M_IGNORE_NAK,
  • I2C_M_NO_RD_ACK, and
  • I2C_M_RECV_LEN.
