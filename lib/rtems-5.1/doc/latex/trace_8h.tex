\subsection{Detailed Description}
\begin{DoxyParagraph}{Purpose}

\end{DoxyParagraph}
Standard output methods for reporting debug information, warnings and errors, which can be easily be turned on/off.

\begin{DoxyParagraph}{Usage}

\begin{DoxyEnumerate}
\item Initialize the D\+B\+GU using T\+R\+A\+C\+E\+\_\+\+C\+O\+N\+F\+I\+G\+U\+R\+E() if you intend to eventually disable A\+LL traces; otherwise use D\+B\+G\+U\+\_\+\+Configure().
\item Uses the \mbox{\hyperlink{trace_8h_a67a8f9bc74358aa21745a76e23768dae}{T\+R\+A\+C\+E\+\_\+\+D\+E\+B\+U\+G()}}, T\+R\+A\+C\+E\+\_\+\+I\+N\+F\+O(), T\+R\+A\+C\+E\+\_\+\+W\+A\+R\+N\+I\+N\+G(), T\+R\+A\+C\+E\+\_\+\+E\+R\+R\+O\+R() T\+R\+A\+C\+E\+\_\+\+F\+A\+T\+A\+L() macros to output traces throughout the program.
\item Each type of trace has a level \+: Debug 5, Info 4, Warning 3, Error 2 and Fatal 1. Disable a group of traces by changing the value of T\+R\+A\+C\+E\+\_\+\+L\+E\+V\+EL during compilation; traces with a level bigger than T\+R\+A\+C\+E\+\_\+\+L\+E\+V\+EL are not generated. To generate no trace, use the reserved value 0.
\item Trace disabling can be static or dynamic. If dynamic disabling is selected the trace level can be modified in runtime. If static disabling is selected the disabled traces are not compiled.
\end{DoxyEnumerate}
\end{DoxyParagraph}
\begin{DoxyParagraph}{trace\+Levels Trace level description}

\begin{DoxyEnumerate}
\item T\+R\+A\+C\+E\+\_\+\+D\+E\+B\+UG (5)\+: Traces whose only purpose is for debugging the program, and which do not produce meaningful information otherwise.
\item T\+R\+A\+C\+E\+\_\+\+I\+N\+FO (4)\+: Informational trace about the program execution. Should enable the user to see the execution flow.
\item T\+R\+A\+C\+E\+\_\+\+W\+A\+R\+N\+I\+NG (3)\+: Indicates that a minor error has happened. In most case it can be discarded safely; it may even be expected.
\item T\+R\+A\+C\+E\+\_\+\+E\+R\+R\+OR (2)\+: Indicates an error which may not stop the program execution, but which indicates there is a problem with the code.
\item T\+R\+A\+C\+E\+\_\+\+F\+A\+T\+AL (1)\+: Indicates a major error which prevents the program from going any further. 
\end{DoxyEnumerate}
\end{DoxyParagraph}
