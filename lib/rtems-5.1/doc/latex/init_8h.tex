\subsection{Detailed Description}
Constants and Structures Associated with the Initialization Manager. 

This include file contains all the constants and structures associated with the Initialization Manager. This manager is responsible for initializing R\+T\+E\+MS, creating and starting all configured initialization tasks, invoking the initialization routine for each user-\/supplied device driver, and initializing the optional multiprocessor layer.

This manager provides directives to\+:


\begin{DoxyItemize}
\item initialize the R\+T\+E\+MS executive
\item shutdown the R\+T\+E\+MS executive 
\end{DoxyItemize}
