\subsection{Detailed Description}
Serial Mouse Driver. 

This file describes the Serial Mouse Driver for all boards. This driver assumes that the B\+SP or application will provide an implementation of the method \mbox{\hyperlink{group__libmisc__serialmouse_ga7103f15e10a857e48f4f026a58968288}{bsp\+\_\+get\+\_\+serial\+\_\+mouse\+\_\+device()}} which tells the driver what serial port device to \mbox{\hyperlink{open_8c_a62e93eb794fed24c0f46c6acda6ea510}{open()}} and what type of mouse is connected.

This driver relies on the Mouse Parser Engine.
