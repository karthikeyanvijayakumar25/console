\subsection{Detailed Description}
Red-\/\+Black Tree Handler. 

The Red-\/\+Black Tree Handler is used to manage sets of entities. This handler provides two data structures. The rbtree Node data structure is included as the first part of every data structure that will be placed on a R\+B\+Tree. The second data structure is rbtree Control which is used to manage a set of rbtree Nodes.
