\subsection{Detailed Description}
\hypertarget{group__RTEMSBSPsARM_Purpose}{}\subsection{Purpose}\label{group__RTEMSBSPsARM_Purpose}
This file provides a basic A\+PI for P\+IO configuration and usage of user-\/controlled pins. Please refer to the \mbox{\hyperlink{board_8h}{board.\+h}} file for a list of available pin definitions.\hypertarget{group__RTEMSBSPsARM_Usage}{}\subsection{Usage}\label{group__RTEMSBSPsARM_Usage}

\begin{DoxyEnumerate}
\item Define a constant pin description array such as the following one, using the existing definitions provided by the \mbox{\hyperlink{board_8h}{board.\+h}} file if possible\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{keyword}{const} \mbox{\hyperlink{struct__Pin}{Pin}} pPins[] = \{PIN\_USART0\_TXD, PIN\_USART0\_RXD\};}
\end{DoxyCode}
 Alternatively, it is possible to add new pins by provided the full Pin structure\+: 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{comment}{// Pin instance to configure PA10 \& PA11 as inputs with the internal}}
\DoxyCodeLine{\textcolor{comment}{// pull-up enabled.}}
\DoxyCodeLine{\textcolor{keyword}{const} \mbox{\hyperlink{struct__Pin}{Pin}} pPins = \{}
\DoxyCodeLine{     (1 << 10) | (1 << 11),}
\DoxyCodeLine{     REG\_PIOA,}
\DoxyCodeLine{     \mbox{\hyperlink{group__SAME70J19__id_ga2981ffe400e0d14c59058b04bbe91860}{ID\_PIOA}},}
\DoxyCodeLine{     \mbox{\hyperlink{pio_8h_a09864e6fb6c55d82aa7d6ca20f9a4dd4}{PIO\_INPUT}},}
\DoxyCodeLine{     \mbox{\hyperlink{pio_8h_aad63041b7acf2a9d1e72d0c61b2c1b63}{PIO\_PULLUP}}}
\DoxyCodeLine{\};}
\end{DoxyCode}

\item Configure a pin array by calling \mbox{\hyperlink{pio_8h_a1bb66b931fa38004c2178936c269fa76}{P\+I\+O\+\_\+\+Configure()}} with a pointer to the array and its size (which is computed using the P\+I\+O\+\_\+\+L\+I\+S\+T\+S\+I\+ZE macro).
\item Change and get the value of a user-\/controlled pin using the P\+I\+O\+\_\+\+Set, P\+I\+O\+\_\+\+Clear and P\+I\+O\+\_\+\+Get methods.
\item Get the level being currently output by a user-\/controlled pin configured as an output using \mbox{\hyperlink{pio_8h_a6226a6942c31e5f561c5e2b4d811f824}{P\+I\+O\+\_\+\+Get\+Output\+Data\+Status()}}. 
\end{DoxyEnumerate}
