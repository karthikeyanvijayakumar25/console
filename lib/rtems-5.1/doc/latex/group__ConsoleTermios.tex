\subsection{Detailed Description}
Console driver for Termios devices. 

In order to use this driver add the following lines to the Makefile.\+am of the B\+SP\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{libbsp\_a\_SOURCES += ../../shared/console-termios-init.c}
\DoxyCodeLine{libbsp\_a\_SOURCES += ../../shared/console-termios.c}
\DoxyCodeLine{libbsp\_a\_SOURCES += console/console-\mbox{\hyperlink{structconfig__s}{config}}.c}
\end{DoxyCode}


Define the console\+\_\+device\+\_\+table and console\+\_\+device\+\_\+count in the console-\/config.\+c file of the B\+SP.
