\subsection{Detailed Description}
Timer for Blackfin. 

This file manages the benchmark timer used by the R\+T\+E\+MS Timing Test Suite. Each measured time period is demarcated by calls to \mbox{\hyperlink{arm_2lpc24xx_2start_2timer_8c_a9ba0ae2fd8880801fb86979a16cd3b2b}{benchmark\+\_\+timer\+\_\+initialize()}} and \mbox{\hyperlink{arm_2lpc24xx_2start_2timer_8c_ab8dc8517e97918d293d78e85b9454855}{benchmark\+\_\+timer\+\_\+read()}}. \mbox{\hyperlink{arm_2lpc24xx_2start_2timer_8c_ab8dc8517e97918d293d78e85b9454855}{benchmark\+\_\+timer\+\_\+read()}} usually returns the number of microseconds since \mbox{\hyperlink{arm_2lpc24xx_2start_2timer_8c_a9ba0ae2fd8880801fb86979a16cd3b2b}{benchmark\+\_\+timer\+\_\+initialize()}} exitted.
