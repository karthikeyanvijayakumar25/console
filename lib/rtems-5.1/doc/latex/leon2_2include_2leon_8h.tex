\subsection{Detailed Description}
This include file contains information pertaining to the L\+E\+O\+N-\/1. The L\+E\+O\+N-\/1 is a custom S\+P\+A\+RC V7 implementation. This C\+PU has a number of on-\/board peripherals and was developed by the European Space Agency to target space applications.

N\+O\+TE\+: Other than where absolutely required, this version currently supports only the peripherals and bits used by the basic board support package. This includes at least significant pieces of the following items\+:


\begin{DoxyItemize}
\item U\+A\+RT Channels A and B
\item Real Time Clock
\item Memory Control Register
\item Interrupt Control 
\end{DoxyItemize}
