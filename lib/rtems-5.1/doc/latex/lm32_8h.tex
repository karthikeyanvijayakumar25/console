\subsection{Detailed Description}
L\+M32 Set up Basic C\+PU Dependency Settings Based on Compiler Settings. 

This file sets up basic C\+PU dependency settings based on compiler settings. For example, it can determine if floating point is available. This particular implementation is specified to the NO C\+PU port.
