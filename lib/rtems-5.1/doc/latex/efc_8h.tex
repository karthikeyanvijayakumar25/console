\subsection{Detailed Description}
\hypertarget{group__RTEMSBSPsARM_Purpose}{}\subsection{Purpose}\label{group__RTEMSBSPsARM_Purpose}
Interface for configuration the Enhanced Embedded Flash Controller (E\+E\+FC) peripheral.\hypertarget{group__RTEMSBSPsARM_Usage}{}\subsection{Usage}\label{group__RTEMSBSPsARM_Usage}

\begin{DoxyEnumerate}
\item Enable/disable flash ready interrupt sources using \mbox{\hyperlink{efc_8h_a3f6311b4fcbc57fe4079b1ef05b3fc14}{E\+F\+C\+\_\+\+Enable\+Frdy\+It()}} and \mbox{\hyperlink{efc_8h_a47e9d8bbb518d8c528cf7317b9080aa3}{E\+F\+C\+\_\+\+Disable\+Frdy\+It()}}.
\item Translates the given address into which E\+E\+FC, page and offset values for difference density flash memory using \mbox{\hyperlink{efc_8h_a170fa534d132c75b2851f0a42f7b71b0}{E\+F\+C\+\_\+\+Translate\+Address()}}.
\item Computes the address of a flash access given the E\+FC, page and offset for difference density flash memory using \mbox{\hyperlink{efc_8h_a50ce7469847193013d20ad0a928bd953}{E\+F\+C\+\_\+\+Compute\+Address()}}.
\item Start the executing command with \mbox{\hyperlink{efc_8h_aa9b30cd8bbb9ff48c04922ad5b36adfb}{E\+F\+C\+\_\+\+Perform\+Command()}}
\item Retrieve the current status of the E\+FC using \mbox{\hyperlink{efc_8h_a112cc6ecce4ee96d5262c40df8fd0ef1}{E\+F\+C\+\_\+\+Get\+Status()}}.
\item Retrieve the result of the last executed command with \mbox{\hyperlink{efc_8h_a5a13faa6909d8d0500679fdaa8245a40}{E\+F\+C\+\_\+\+Get\+Result()}}. 
\end{DoxyEnumerate}
