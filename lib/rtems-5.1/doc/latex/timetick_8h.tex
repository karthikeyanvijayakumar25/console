\subsection{Detailed Description}
\begin{DoxyParagraph}{Purpose}

\end{DoxyParagraph}
Methods and definitions for Global time tick and wait functions.

Defines a common and simplest use of Time Tick, to increase tick\+Count every 1ms, the application can get this value through Get\+Tick\+Count().

\begin{DoxyParagraph}{Usage}

\end{DoxyParagraph}

\begin{DoxyEnumerate}
\item Configure the System Tick with Time\+Tick\+\_\+\+Configure() when M\+CK changed \begin{DoxyNote}{Note}
Must be done before any invoke of Get\+Tick\+Count(), Wait() or Sleep().
\end{DoxyNote}

\item Uses Get\+Tick\+Count to get current tick value.
\item Uses Wait to wait several ms.
\item Uses Sleep to enter wait for interrupt mode to wait several ms. 
\end{DoxyEnumerate}
