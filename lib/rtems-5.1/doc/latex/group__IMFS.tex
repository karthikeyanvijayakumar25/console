R\+T\+E\+MS load tarfs. 

This file implements the \char`\"{}mount\char`\"{} procedure for tar-\/based I\+M\+FS extensions. The T\+AR is not actually mounted under the I\+M\+FS. Directories from the T\+AR file are created as usual in the I\+M\+FS. File entries are created as I\+M\+F\+S\+\_\+\+L\+I\+N\+E\+A\+R\+\_\+\+F\+I\+LE nodes with their nods pointing to addresses in the T\+AR image.

Here we create the mountpoint directory and load the tarfs at that node. Once the I\+M\+FS has been mounted, we work through the tar image and perform as follows\+:
\begin{DoxyItemize}
\item For directories, simply call \mbox{\hyperlink{mkdir_8c_aee98bbe743c2d14dbaa67f01c3fb9ed5}{mkdir()}}. The I\+M\+FS creates nodes as needed.
\item For files, we make our own calls to I\+M\+FS eval\+\_\+for\+\_\+make and create\+\_\+node.
\end{DoxyItemize}
