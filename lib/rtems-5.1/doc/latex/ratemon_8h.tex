\subsection{Detailed Description}
This include file contains all the constants, structures, and prototypes associated with the Rate Monotonic Manager. This manager provides facilities to implement threads which execute in a periodic fashion.

Directives provided are\+:


\begin{DoxyItemize}
\item create a rate monotonic timer
\item cancel a period
\item delete a rate monotonic timer
\item conclude current and start the next period
\item obtain status information on a period
\item obtain the number of postponed jobs 
\end{DoxyItemize}
