\subsection{Detailed Description}
Provide printf() P\+R\+Ixxx Constante Beyond Standards. 

This include file defines P\+R\+Ixxx constants beyond those in the C and P\+O\+S\+IX standards. These are used to write portable printf() format strings for P\+O\+S\+IX and R\+T\+E\+MS types not in $<$\mbox{\hyperlink{inttypes_8h}{inttypes.\+h}}$>$
